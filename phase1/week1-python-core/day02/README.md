# Day 2 - Student Grade Manager

A Python CLI app to manage student grades using lists and dictionaries.

## Features
- Add students with name, grade, subject
- View all students
- Calculate average grade
- Find top student
- Filter students by minimum grade

## What I learned
- Lists - append, remove, sort, slicing
- List comprehensions
- Tuples - immutable, unpacking
- Dictionaries - CRUD, .get(), .items()
- Sets - union, intersection, difference
- lambda functions with max()
- While loop menu system
