# Day 1 - Profile Card Generator

A Python CLI app that takes user input and displays 
a formatted developer profile card.

## What I built
- format_card() function with type hints
- validate_inputs() with error handling
- User input with input()
- List comprehension for skills parsing

## What I learned
- Variables and data types (int, float, str, bool, None)
- String methods (.strip(), .split(), .join(), .upper())
- f-strings for formatting
- Type hints (name: str, age: int)
- try/except for error handling
- Immutability of strings

## How to run
```bash
python profile_card.py
```
